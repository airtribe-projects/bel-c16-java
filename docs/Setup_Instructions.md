Setup Instructions
# Prerequisites
'Java Development Kit (JDK)': Version 17 or higher is recommended.

'IDE': Visual Studio Code (with Java Extension Pack) or IntelliJ IDEA.

# How to Run
Clone this repository to your local machine.

Open your terminal and navigate to the 'src' folder:

'cd LearnTrack/src'

## Compile the project:

'javac com/airtribe/learntrack/Main.java'

## Run the application:

'java com.airtribe.learntrack.Main'

# Project Features
Interactive CLI Menu.

Automatic ID generation.

Dynamic student storage using ArrayLists.
